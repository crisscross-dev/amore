# Grades Import Template

This folder contains a sample file for importing grades.

- File: `grades_template.csv`
- Columns:
  - `lrn` (student LRN; 12 digits)
  - `subject_code` (e.g., MATH7, SCI7)
  - `term` (e.g., Q1, Q2, Midterm, Final)
  - `grade_value` (0-100; decimals allowed)
  - `remarks` (optional)

Open in Excel and Save As `.xlsx` if needed. Keep the header row intact.
